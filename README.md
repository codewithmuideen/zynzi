# Zynzi

> **Bored? Post a vibe. Feel human again.**

Zynzi is a social spontaneity app that connects real people for impromptu meetups — dinners, walks, games, coffee. No swiping. No awkward DMs. Just real moments, nearby, right now.

Launching in **London & Sheffield**.

---

## Tech Stack

- **React 18** + **Vite**
- **Tailwind CSS**
- **React Router v7**
- **Lucide React** (icons)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── assets/              # Images and static files
├── components/
│   ├── layout/          # Navbar, Footer
│   ├── sections/        # Hero, Features, HowItWorks, etc.
│   └── ui/              # Button, WaitlistForm, SectionLabel
├── constants/           # App-wide copy and config
├── hooks/               # useScrollReveal, useWaitlistForm
└── pages/               # Home, Contact, PrivacyPolicy, TermsOfUse
```

## Key Features

- Spontaneous event discovery and hosting
- Trust-first safety system (public-first events, selfie verification, review scores)
- Interest-matched events (coffee, walks, games, dinner)
- Venue/business host dashboard
- Waitlist with 500+ early signups
